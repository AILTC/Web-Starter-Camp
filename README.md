# Web Starter Camp Example

此專案為 Express + Bootstrap + jquery 的簡單範例。

| On The Server | On The Client  | Development  |
| ------------- | -------------- | ------------ |
| Express       | Bootstrap      | Nodemon      |
|               | jquery         |              |

## 安裝

### 1.安裝 Node.JS

[可由此下載](https://nodejs.org/dist/v8.11.3/node-v8.11.3-x64.msi)安裝檔並安裝。

### 2.安裝相依套件
```
npm i
```

### 3.啟動服務
```
node app.js
```
or
```
npm start
```
